### Explorador de Jogos com API ###

🟢 Este projeto apresenta uma conexão entre back-end e front-end a partir de duas máquinas, acessando uma API local de jogos dos gêneros:

- Ação 🧨;
- Terror 👻;
- Survival 🪓;
- Multiplayer 🤜🤛;
- Indie 🗻;
- RPG ⚔️.
